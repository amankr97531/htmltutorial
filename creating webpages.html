We can create HTML webpages in any of the editor notepad wordpad etc. But the extension name is .html during saving the HTML document we select all files option in the file type in save dialog box then put our name as filename.html

  • Running The HTML Document

  • Attribites of body tag

  1. BGCOLOR:-

    2. This attribute is used to the set of the background color of the webdocument. 
    <BODY BGCOLOR="RED" > </BODY>
    We can also set the 16 color as red, green,blue,yellow,pink,etc

  2. Text Color

    3. The attribute is used to the set of the text color display on the webpages. 
      <BODY> BGCOLOR="RED" > </BODY>

  3. Link

    4. This attribute is used to set the link color display on the webpages.

    V LINK (Visited Link)
    A Link (Anchor link)

    This two attribute is used to set Visited link and Anchor link displayed on the webpages.
    NOTE :-
    All the above attribute may set in a single body syntax seperated with blankspace.

  4. Background

    5. This attribute is set is used to set background image on the webpages. 
      <BODY> BACKGROUND="PATH\FILENAME.JPEG" </BODY>

  5. Heading Tag

    6. This attribute is the tag of format text into heading there are six type of the heading predefined in html. 
    <HTML>
<TITLE> HEADING </TITLE>
<HEAD>
<H1> THIS IS HEADING </H1>
<H2> THIS IS HEADING </H2>
<H3> THIS IS HEADING </H3>
<H4> THIS IS HEADING </H4>
<H5> THIS IS HEADING </H6>
<H1> THIS IS HEADING </H1>
</HEAD>
</HTML>

  6. Paragraph

    7. This tag is used to set the paragraph in html webpages.
      <P> PARAGRAPH CONTENT </P>

  7. Break Tag

    8. This tag is self ending tag used to break the line in html web documents. 
    </br>

  8. Formatting Tag

    9. This tag is used to display the content in preformat.
    <PRE> MESSAGE CONTENT </PRE>

  9. Adress tag

    10. This tag is used to display the address in predefined formation the webpages.

    Other Formatting Tag

    •  <center>

      This tag is used to align the text/content in center margin. 
       <BODY> 
 <CENTER> MESSAGE CONTENT </CENTER>
</BODY>  

    •  <Small>
      • This tag is used to convert the text into small text. 

    •  <U>

      This tag is used to underline the text. 

    •  <I>

      This tag is used to convert the italic. 

    •  <Strike>

      This tag is used to strike the content. 

    •  <Sub>

      This tag is used to convert the text into the subscript. 

    •  <Sup>

      This tag is used to convert the text into superscript. 

    •  <Big>

      This tag is used to convert the content into capital form.
