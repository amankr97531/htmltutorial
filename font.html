FONT

It is the tag which is used to change the font of the text information display on the web browser font tag has multiple attribute through which font is controlled on the browser.

  • Color

    • This attribute is used to set the font color of the text. 
          < font color="red" >.........</font>

  • Size

    • This attribute is set the size of the text display on the screen. 
          < font size="5" >.........</font>

  • Face

    • This attribute is used to set the name of the font in html. 
          < font face="Arial" >.........</font>
