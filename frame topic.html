FRAME

Frame is the facility in HTML to divided the screen in multiple part so that multiple web document may display on the browser screen at once we have two important tag to divided the screen.

  • Frameset

    • This is the tag to define how many parts the screen can divided either vertically or horizontally.

    The fame division may be done either in pixel or in percentage relative position of the frame.

    <Frameset rows="size of the first frame,size of the second frame"/>
    <Frameset rows="200,300,400">..............</frameset>

    <Frameset rows="10%,20%,70%">..............</frameset>
    <Frameset rows="20%,40%">..............</frameset>

    Each frame is define with the frame tag to display content as webpages.
    Src is the attribute to define the html page to display in that frame .
    <frame src="sample.html"/>

  • Other attribute of the frame is

    1. Scrolling:- yes or no
    2. Frameborder:- It may be 0 or 1
    3. Noresize:- Fix size of the frame control extend at the runtime
    <Frameset="rows="10%,70%,20%">
    <frame src="sample1.html>
    <frame src="sample1.html>
    <frame src="sample1.html>

    In the similar manner the distribution of frame column wise mode may be done through cells attribute.

    <Frameset="cols="10%,70%,20%">
    <frame src="sample1.html>
    <frame src="sample1.html>
    <frame src="sample1.html>
    </frameset>

  • Other attribute of the frame

    • Name attribute

    This attribute is use to define name of the frame.

    Additional attribute of anchor tag

  • Target attribute

    • This attribute is use to set the target frame where the information page to be displayed.
    <a href="sample.html" target="frame name>.........</a>
