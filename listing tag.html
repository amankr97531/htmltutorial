Listing Tag

Listing tag is used create the list of items in the html.
There are three types of listing tag in html

  1. Ordered List

    This tag is used to create the order list where ordering is controlled by the type tag.

    if type= i - i , ii , iii
    if type= a - a , b , c , d
    if type= A - A , B , C , D
    if type= 1 - 1 , 2 , 3 , 4

    The starting number of order is controlled by start attribute.

     <ol type=1>
    
.........
    
.........
    
.........
    
</ol>
    The item of the list is controlled by 
     <li> tag 
        <ol type=1>
         <li> monitor </li>
         <li> mouse </li>
         <li> keyboard </li>
         <li> cpu </li>
         </ol>
    This is the order list may be run in same manner.

  2. Unordered List

    This is the html tag which is used to create the unordered list in html. 
     <ul>
    It is the html tag to create unordered list here. The bullet of the list is controlled ny type attribute and item of the list is given with listing tag.

    The value of the type may be square , disc , circle.

  3. Definition list

    4. This is a type of list that create topic and their definition over the html webpage there are three tag which is used to create definition list in html.
  •  <dl>
    This tag is used to create the difinition list.
  •  <dt>
    This tag is used to define definition topic. 
  •  <dd>
    This tag is used to define the topic given in dt tag. 
     <dl>
<dt> ........ </dt>
<dd>

</dd>
</dl>
