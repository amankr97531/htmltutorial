MARQUEE TAG

This is the tag which is use to display the text or picture content scrolling on the screen either on the horizontally or vertically.

  • Behaviour

    • It is a motion type may be alternate scroll or slide.

  • Bgcolor

    • This attribute is sue to set the background of the panel on which text content will be scrolled.

  • Direction

    • This attribute is use to set the height and width of the panel height and width may be in inch pixel or relative according to parent window.

  • Loop

    • This attrbute is use to scrolling the text number of times.

  • Scroll delay

    • It is the attribute to set the delay to display text under scrolling.

    Syntax
        <marquee height="100" width="400" bgcolor="blue" direction="right" behaviour="scroll"/>............</marquee>
