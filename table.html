TABLE

Table is the facility in a HTML to arganize the data in rows and columns. 
  • <Table>
    • It is the tag to create table. 
  • <Tr>
    • It is the tag to create rows of the table. 
  • <Td>
It is the tag which is used to create a column of the table.
It means if complete table is constructed by three tag. 
 <Table> , <Tr> , <Td> and fourth tag <Th>
Which is just like td used to create the column heading Each tag must be open and close to get corresponding affect. 
 <Table> 
       <Tr>
	   <Th>.............</Th>
	   <Td>.............</Td>
	   <Td>.............</Td>
	   </Tr>
	   </Table>

  • Caption tag

    • This tag is used within the table tag to provide heading to the table.

  • Attributes of the captions

  1. Align

To set the caption either at the bottom or top left center right. 
         <Table>
         < caption align="bottom">
         student record
         </caption>

  • Attributes of the table

    1. Border

      2. This attribute is use to set the border of the table in pixel.

    2. Cellspacing

      3. This attribute is use to set the number of pixel spaces between the cell.

    3. Cellpadding

      4. This attribute is use to set of the space within each cell to display the content.

    4. width

      5. It is use to set of the width of the table either in pixel or percentage (static and relative in percentage).

    5. Height

    It is use to set the height of the table either in pixel or relative in percentage. 
             < table border="2" cellspacing="3" cellpadding="4" width="40%" heigth="70%" align="center">

  • Attribute of the <Th> and <Td>

  1. Width

    2. It is use to set width of the column in the table it may be also in pixel or percentage.

  2. Align

    3. It is use to align the content of the cell either in left right or center.

  3. Rowspan

    4. It is use to merge the cell vertically that is number of rows in a particular column to be merged for a particular cell.

    <Th rowspan="2">...........</Th>
    <Td rowspan="3">...........</Td>

  4. Colspan

    5. It is use to merge the number of column horizontally to make a single column.

    <Th colspan="2">...........</Th>
    <Td colspan="3">...........</Td>

  5. Bgcolor

    6. It is use to set the background color of the cell.

  6. Color

    7. It is use to set text color of the cell.
